# es-example
ECMAScriptの新機能まとめ

# ディレクトリ構造

```
.
├── input
│   └── esXXXX
│       └── ...
├── output
│   └── esXXXX
│       └── ...
└── com.sh

```

|ディレクトリ名|説明|
|:-:|:--|
|input/esXXXX/...|ECMAScriptXXXXによる新機能のコード|
|output/esXXXX/...|ECMAScriptXXXXで記述されたコードをBabelでトランスパイルしたもの|
|com.sh|トランスパイル用のシェル|
